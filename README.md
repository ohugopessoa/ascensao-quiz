# Ascensão — Quiz Diagnóstico

Diagnóstico personalizado de interferências instaladas que bloqueiam a prosperidade. Construído a partir das respostas do usuário, revela origem, padrão e mecanismo do que trava.

Single-page application em HTML/React/Babel standalone. Sem build step.

## Rodar localmente

```bash
npx http-server . -p 8080
# abre http://localhost:8080
```

## Deploy

Hospedado via [Vercel](https://vercel.com) — deploy automático a cada push na `main`.

## Estrutura

- `index.html` — aplicação inteira (quiz + diagnóstico)
